## Nội dung dự kiến

```
postman/
├── collections/       # *.postman_collection.json
├── environments/      # *.postman_environment.json
└── data/              # CSV/JSON cho data-driven testing
```

## Cách dùng (khi có file)
1. Mở Postman → Import các file trong `collections/` và `environments/`.
2. Chọn environment (vd `local`).
3. Chạy request hoặc dùng Collection Runner với file trong `data/`.
